Climate change involves global warming and other side effects such as increased melting glaciers, rainstorms, sea level, and droughts. This project focuses on global warming which climate.gov specifically defines as "Earth's rising surface temperature."


This project analyzes changes in global surface temperatures across time. Using data from NASA's GISS Surface Temperature Analysis and other sources, this project consists of machine learning models, an interactive map, and various charts that depict increasing global temperature trends.
